# Geek movies
GeekMovies es una pagina web enfocada para los fans de los superheroes, en esta encontraras las peliculas mas recienes, las recomendaciones, podras añadirlas a tu propia lista.
Este proyecto está enfocado en practicar:
- HTML semántico
- CSS responsivo y diseño moderno
- JavaScript para funcionalidades interactivas
- ## Tecnologías utilizadas

- **HTML5**
- **CSS**
- **JavaScript**
